## Featured Project

<br/>

<div align="center">

### [codetopodcast.me](https://codetopodcast.me)

`FastAPI` &nbsp;·&nbsp; `React JS` &nbsp;·&nbsp; `Google ADK` &nbsp;·&nbsp; `Azure` &nbsp;·&nbsp; `Vercel`

<br/>

> A SaaS platform that transforms any code repository into a structured, listenable podcast using Agentic AI.
> It handles end-to-end pipeline from repo ingestion and topic extraction, to script generation and audio synthesis.
> Built and deployed solo as a live production product.
</div>

**→ What it does:** Generates a podcast from a GitHub repository link. 

**→ Who it's for:** Senior developers, junior developers, and students.

**→ What makes it unique:** It is designed to go in depth of the code, and catch key connection and flows within the code, while keeping the API costs efficient.
